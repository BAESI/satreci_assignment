## 주요 기능
* 학생 정보 등록
* 학생 정보 삭제
* 학생 목록 출력
  * 이름순 출력
  * 학번순 출력
  * 평균순 출력

## 실행 방법
현재 cmake-build-debug 디렉토리안에 studentlist.txt 파일안에 기본데이터가 저장되어 있는 상태입니다.<br> 
프로그램을 실행시키기 위해 cmake-bulid-debug 디렉토리에서 ./satrci 명령어를 통해 프로그램을 실행할 수 있습니다.
